# 贾瑶瑶 · BMS HIL 测试工程师个人主页

这是一个无需后端、无需构建工具的静态个人简历网站，适合直接部署到 GitHub Pages。

## GitHub Pages 部署

1. 在 GitHub 新建一个公开仓库。
2. 将本文件夹中的全部文件上传到仓库根目录。
3. 打开仓库的 `Settings` → `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/ (root)`，保存。
6. 等待一两分钟，GitHub 会显示可访问的网址。

## 本地预览

直接双击 `index.html` 即可预览。为保证所有浏览器功能一致，也可以在本目录启动任意静态文件服务器。

## 文件说明

- `index.html`：页面内容
- `styles.css`：视觉样式与响应式布局
- `script.js`：主题切换、项目筛选、展开折叠、数字动效等交互
- `assets/`：PDF 简历文件
